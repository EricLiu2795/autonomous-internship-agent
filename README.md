# Autonomous Internship Agent 🚀

An AI-powered platform that helps college students automate and optimize their internship application process.

## Features

- **Smart Application Strategy** - AI-generated personalized internship hunting strategies
- **Company Matching** - Get matched with top companies based on your profile
- **Resume Optimization** - Tailored resume bullets for each application
- **Outreach Automation** - Pre-written cold outreach templates
- **Pipeline Tracking** - Visual Kanban board to track application progress
- **Follow-up Reminders** - Automated timeline for follow-ups

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
